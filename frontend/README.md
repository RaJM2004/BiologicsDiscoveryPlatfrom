# Biologics Discovery Platform - Frontend

This repository contains the frontend user interface for the Biologics Discovery Platform.

## Tech Stack
- **Structure**: HTML5, Jinja2
- **Styling**: CSS3 (Vanilla + Custom Design System)
- **Logic**: Vanilla JavaScript
- **Visualization**: Plotly, Chart.js, Mol/NGL Viewer (3D Protein Vis)

## Folder Structure
- `templates/`: HTML templates (Jinja2 compatible)
- `static/`: Static assets
  - `css/`: Stylesheets
  - `js/`: Client-side scripts
  - `assets/`: Images and icons

## Usage
These templates are designed to be served by the Backend application (FastAPI) or can be adapted for a standalone static site if needed.
